什么是缓存雪崩：

redis是不可能把所有数据都永久缓存起来的（内存昂贵有限），所以Redis需要对数据设置过期时间，并采用的是懒惰删除和定期删除的策略。如果缓存数据设置的过期时间是相同的，并且Redis恰好将这部分数据删除了。这会导致在这段时间内，这些缓存失效，全部请求到数据库中。这就是缓存雪崩。
